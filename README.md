# block-blue-check

트윗 상세 페이지에서 **원문 트윗 멘션 + 관련 트윗 범위의 계정 중 블루 체크 계정**을 버튼 한 번으로 차단하는 크롬 익스텐션입니다.

## 기능

- `x.com/<user>/status/<id>` 또는 `twitter.com/<user>/status/<id>` 페이지에서 동작
- 원문 트윗 본문의 `@멘션` 수집
- 관련 트윗 영역의 작성자 및 해당 트윗 내 `@멘션`도 수집
- `블루체크 멘션 차단` 버튼 클릭 시 화면 내 트윗 카드 기준으로 순차 실행
- 처리 결과(차단 성공/실패/이미차단/비파딱 스킵) 표시

## 설치 방법 (개발자 모드)

1. 크롬에서 `chrome://extensions` 진입
2. 우측 상단 `개발자 모드` 활성화
3. `압축해제된 확장 프로그램을 로드합니다` 클릭
4. 이 저장소 루트 폴더(`block-blue-check`) 선택
5. 확장 목록에 `Block Blue Check Mentions`가 표시되면 완료

## 사용 방법

1. X(트위터)에 로그인된 상태에서 트윗 상세 페이지로 이동
2. 원문 트윗 하단 액션 영역 근처의 `블루체크 멘션 차단` 버튼 클릭
3. 순차 차단 프로세스가 시작되고 상태 메시지로 결과 확인

## 동작 원리 요약 (UI 자동화)

- 콘텐츠 스크립트가 트윗 상세 페이지를 감지
- 원문 트윗의 `@멘션` 목록을 추출
- 관련 트윗들의 작성자/멘션 계정을 함께 추출
- 화면에 보이는 트윗 카드에서 블루 체크 배지를 확인
- 해당 카드의 `더보기` 메뉴에서 `차단`을 클릭하고 확인 버튼을 자동 클릭

## 제한사항 및 주의

- X 웹 UI의 DOM/라벨에 의존하므로, UI 업데이트나 언어 설정 변화 시 동작이 깨질 수 있습니다.
- 현재 방식은 **화면에서 메뉴를 열 수 있는 계정(보이는 트윗 카드 작성자)** 중심으로 안정적으로 동작합니다.
- 화면에 보이지 않는 계정은 차단 대상에 포함돼도 실제 실행에서 제외될 수 있습니다.
- 자동 차단 기능은 계정 운영 정책/약관 측면의 리스크가 있을 수 있으니 본인 책임 하에 사용하세요.
- 이미 차단된 계정은 `이미차단`으로 집계됩니다.

## 빠른 점검 체크리스트

- 원문 멘션/관련 트윗 대상이 모두 없을 때 안내가 뜨는지
- 보이는 트윗 카드 기준으로 블루 체크가 없을 때 `비파딱 스킵`이 증가하는지
- 블루 체크 대상(원문+관련)이 여러 명일 때 순차 처리되는지
- 일부 계정에서 메뉴 탐지 실패 시 `실패` 집계와 예시 오류가 표시되는지
